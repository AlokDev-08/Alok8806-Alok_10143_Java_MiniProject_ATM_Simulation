# ATM Simulation (Java OOP Project)

This is a simple ATM Simulation project developed using Java. The main purpose of this project is to understand how basic banking operations work using Object-Oriented Programming (OOP) concepts.

---

## Features

- PIN-based authentication
- Withdraw money 
- Check account balance
- Simple menu-driven interface
- Handles invalid inputs and errors

---

## Technologies Used

- Java (Core Java)
- OOP Concepts (Classes, Objects, Encapsulation)
- Exception Handling
- Scanner (for user input)

---

## Project Structure

- `ATM` → Handles user authentication  
- `Account` → Manages balance and transactions  
- `Main` → Controls program flow  

---

## How to Run

1. Install Java (JDK)
2. Open the project in any IDE (VS Code / Eclipse)
3. Compile the program:
   ```bash
   javac ATM.java
